📝 HOW IT WORKS NOW:

The app tries 4 different methods to detect IP:
  1. expoConfig.hostUri (Expo SDK 46+)
  2. manifest2.extra.expoGo.debuggerHost (SDK 50+)
  3. manifest.debuggerHost (older versions)
  4. Constants.debuggerHost (fallback)

If all fail:
  → Uses manual IP: 10.69.76.87
