📊 HOW DATA STORAGE WORKS:

When you signup:

  Phone App
     ↓
  Sends: {name, email, password, role}
     ↓
  Backend API (port 3000)
     ↓
  Validates & Hashes Password
     ↓
  PostgreSQL Database
     ↓
  Stores in 'users' table:
  ┌─────────────────────────────┐
  │ id  │ name  │ email  │ role │
  ├─────────────────────────────┤
  │ 1   │ Svce  │ svce@  │ mgmt │
  │ 2   │ Test  │ test@  │ acct │
  └─────────────────────────────┘
     ↓
  Returns: Success + JWT Token
     ↓
  Phone Shows: "Welcome, Svce!"
