  Run: setup-ip.bat
  
  Or manually:
  Run: ipconfig
  Look for: IPv4 Address (e.g., 192.168.1.100)

STEP 2: Update API Configuration
  Edit: src\config\api.js
  
  Change line 6:
  const API_BASE_URL = 'http://YOUR_IP:3000/api';
  
  Replace YOUR_IP with your actual IP

STEP 3: Start Backend (if not running)
  cd backend
  npm run dev
  
  Should see: "Server running on port 3000"

STEP 4: Restart Expo App
  In Expo terminal, press: r
  
  Or restart:
  npx expo start

🔧 TROUBLESHOOTING:

Can't connect?
  1. Check backend is running: npm run dev
  2. Check IP in api.js is correct
  3. Both on same WiFi network
  4. Firewall allows port 3000

Backend not starting?
  1. Database connected? (port 5433)
  2. Password correct? (admin123)
  3. Run: npm run setup

App crashing?
  1. Clear cache: npx expo start -c
  2. Check: TEST-GUIDE.md
