📸 PRODUCT SKETCH OPTIONS:

When adding sketch, choose from:

1. 📷 CAMERA
   • Take photo directly
   • Instant capture
   • Edit before saving

2. 🖼️ GALLERY
   • Select from phone gallery
   • Choose existing photo
   • Edit before saving

3. 🔗 URL
   • Enter image URL
   • Use online images
   • Quick option

📝 BULK BOM ENTRY:

Add multiple materials at once!

Format (one per line):
  Material Name, Quantity, Unit

Example:
┌─────────────────────────────────────┐
│ Steel Sheet, 50, kg                 │
│ Plastic Components, 100, pcs        │
│ Paint, 5, l                         │
│ Screws, 200, pcs                    │
│ Rubber Gasket, 10, m                │
└─────────────────────────────────────┘

Click "Add All" → All 5 materials added!

🔄 WORKFLOW:

NPD User:
  1. Opens project from dashboard
  2. Views full details
  3. Updates status
  4. Adds sketch:
     - Take photo with camera, OR
     - Select from gallery, OR
     - Enter URL
  5. Adds materials (bulk):
     - Enters all materials at once
     - Format: Name, Qty, Unit
     - Clicks "Add All"
  6. Reviews BOM list
  7. Deletes if needed

Management:
  • Sees all updates
  • Views sketch
  • Reviews BOM
  • Tracks progress
