✨ FEATURES:

Create Project:
  • Click "+ New"
  • Enter project name
  • Add description
  • Set priority (Low/Medium/High/Urgent)
  • Optionally assign to NPD

Assign Project:
  • Select NPD user from dropdown
  • Project assigned instantly
  • Status updates to "in_progress"

View Projects:
  • See all company projects
  • Color-coded priority
  • Status badges
  • Assignment info
