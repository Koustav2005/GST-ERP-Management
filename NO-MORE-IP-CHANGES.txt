😫 THE PROBLEM:

Your IP changes when:
  • You reconnect to WiFi
  • Laptop restarts
  • Router assigns new IP
  • You switch networks

Then you have to:
  • Find new IP
  • Update api.js
  • Restart app
  • Annoying! 😤

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ SOLUTION 1: AUTO-UPDATE SCRIPT (EASIEST)

Just run this when you open your laptop:

  auto-update-ip.bat

What it does:
  1. Finds your current IP automatically
  2. Updates src/config/api.js
  3. Shows the new IP
  4. Done! ✨

Usage:
  1. Open laptop
  2. Double-click: auto-update-ip.bat
  3. Start backend: npm run dev
  4. Start frontend: npx expo start
  5. Press: r (to reload)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ SOLUTION 2: SMART AUTO-DETECTION (BEST!)

Use the smart config that detects IP automatically!

Step 1: Rename files
  Rename: src/config/api.js → api-manual.js
  Rename: src/config/api-smart.js → api.js

Step 2: Restart app
  npx expo start
  Press: r

That's it! IP is detected automatically! 🎉

How it works:
  • Expo knows your computer's IP
  • Smart config reads it automatically
  • No manual updates needed
  • Works every time!

✅ SOLUTION 3: SET STATIC IP (PERMANENT FIX)

Set your IP to never change:

Step 1: Open Network Settings
  Press: Win + R
  Type: ncpa.cpl
  Press: Enter

Step 2: Configure WiFi
  Right-click: WiFi adapter
  Select: Properties
  Double-click: Internet Protocol Version 4

Step 3: Set Static IP
  ○ Use the following IP address:
  
  IP address: 192.168.1.100
  Subnet mask: 255.255.255.0
  Default gateway: 192.168.1.1
  
  Preferred DNS: 8.8.8.8
  Alternate DNS: 8.8.4.4

Step 4: Update Config
  Edit: src/config/api.js
  Change to: http://192.168.1.100:3000/api

Done! IP never changes again! 🎯
