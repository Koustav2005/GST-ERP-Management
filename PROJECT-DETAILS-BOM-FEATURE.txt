🎨 BILL OF MATERIALS:

Add Material Form:
  • Material Name *
  • Quantity * (number)
  • Unit (pcs, kg, l, m, units)
  • Estimated Cost (₹)
  • Supplier (optional)

Example:
┌─────────────────────────────────────┐
│ Steel Sheet                         │
│ 50 kg • ₹5000                       │
│ Supplier: ABC Steel                 │
│                              [🗑️]   │
├─────────────────────────────────────┤
│ Plastic Components                  │
│ 100 pcs • ₹2000                     │
│                              [🗑️]   │
├─────────────────────────────────────┤
│ Estimated Total Cost:    ₹7,000.00 │
└─────────────────────────────────────┘
