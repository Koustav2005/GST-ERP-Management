🚀 SETUP IN 3 STEPS:

STEP 1: Setup Database
  Open Command Prompt in: gst-management-app\backend
  Run: setup.bat
  Choose option 1 and enter your PostgreSQL password

STEP 2: Start Backend
  Open Command Prompt in: gst-management-app\backend
  Run: start.bat
  Keep this window open!

STEP 3: Start Frontend
  Open NEW Command Prompt in: gst-management-app
  Run: start-frontend.bat
  Scan QR code with Expo Go app
